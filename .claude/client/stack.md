# Stack

- React 18.3 + TypeScript 5.8, Vite 6.3 (SWC), Mantine UI 8.3
- DeepSignal (via `deepsignal/react`) for reactive state management
- React Router DOM 7.6 with lazy-loaded routes and permission-based access
- Firebase 12 (Auth + Storage) for authentication and file uploads
- Stripe (`@stripe/react-stripe-js` + `@stripe/stripe-js`) for payments
- Socket.io-client for real-time features
- Tabler Icons, Recharts, dayjs, lodash, react-to-print
- Path alias: `@/*` → `./src/*`
